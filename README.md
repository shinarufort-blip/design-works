# バナー広告デザイン制作

温泉旅館向けのバナー広告をPhotoshop・Adobe Fireflyを使用して制作しました。
「特別な時間」をテーマに、秋の紅葉と温泉の雰囲気を組み合わせたビジュアルです。

## 制作の意図・工夫した点

- 夜の温泉と紅葉という「非日常感」を前面に出し、旅行意欲を高めるビジュアルに
- 「特別な時間を、あなたへ。」というコピーで感情に訴えるメッセージ設計
- テキスト上配置・下配置の2パターンを制作し、掲載媒体に合わせて使い分けられるよう対応
- 無料券バナーは「入浴一回無料券」を右側に配置し、特典を視覚的に強調
- Adobe Fireflyを活用して背景画像を生成・合成

## 使用ツール

- Adobe Photoshop
- Adobe Firefly（AI画像生成）
